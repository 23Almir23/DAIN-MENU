## Kritische Regeln (niemals ändern)

1. **1 User = 1 Restaurant** — kein Multi-Tenant
2. **toDb() / toClient()** auf jeder Route
3. **resolveTranslation()** immer mit explizitem `baseLang` — nie `"en"` hardcoden
4. **AI ist assistiv** — `needsReview = true` auf jedem AI-Output
5. **`/guest?r={restaurantId}`** permanent — QR-Codes weltweit
6. **Englisch ist NICHT die Default-Sprache** — `restaurant.baseLanguage` ist die Quelle
7. **Drizzle ORM only** — kein Raw SQL
8. **Credit-System → Migration im Sprint** — Code hat noch Credits, werden auf Unlimited AI (€2.50 flat) migriert. Keine Credit-Spalten löschen ohne explizite Migration.
