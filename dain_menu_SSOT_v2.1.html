Die wahre Vision

Dain Menu ist der Beweis, dass ein Solo-Founder mit einer AI-Mannschaft eine globale AI/SaaS aufbauen kann — schneller, günstiger, besser als ein 10-Personen-Team. Gebaut mit Exit- oder Partial-Exit-Readiness für strategische Acquirer (Google, Booking.com, Toast, PE).

Für Restaurants

  • Globaler AI-Partner für mehrsprachige Karten
  • Scan Dish: Foto → 10 Sprachen in <5 Min
  • AI-Companion "Kai" — proaktiv, kennt das Restaurant
  • Von Pain (Sprachbarriere) zu Gain (Umsatz)

R01
toDb() / toClient() auf allen Routen
Jedes neue Feld, jede neue Route. ESLint-Lint-Rule aktiv ab Sprint Tag 1.
R02
resolveTranslation() immer mit explizitem baseLang
Nie 3 Argumente. Nie "en" hardcoded. Immer restaurant.baseLanguage.
R03
needsReview = true auf ALLEN AI-Outputs
Text, Übersetzung, Allergene, Kalorien, Photo Enhancement, Dynamic Pricing. Keine Ausnahme.
R04
/guest?r={restaurantId} URL ist permanent
Gedruckte QR-Codes weltweit brechen sonst. Nie ändern.
R05
1 User = 1 Restaurant (bis Phase 3)
Kein Team-Invite, kein Shared Access, kein Multi-Restaurant.
R06
Englisch ist nicht die Default-Sprache
DEFAULT_LANGUAGE = UI-Fallback. restaurant.baseLanguage = echte Sprache.
R07
Nur drizzle-kit generate → drizzle-kit push
Nie npm run db:push, nie manuelles SQL, nie Migration-Files editieren.

Personality

Precise. Worldly. Effortless.

Positionierung

Toast/Square = POS-Hardware. Dain Menu = Premium-Concierge. 5 Min von Upload zu globalem QR-Code.

Accent Color

#c9703a Terrakotta. International, erinnert an Essen, nicht Tech. Einzige Akzentfarbe.

Typography

DM Sans Headline+Body. DM Mono für Zahlen/Codes. Keine Serifs (CJK/Arabic inkompatibel).

Visual World

Editorial Foodfotografie. AI Enhancement Vorher/Nachher. Smartphone auf Restauranttisch. Kein Stock.
